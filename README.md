# MongoClient

## Enviroment using docker 🐋

```
docker-compose build
docker-compose up -d

docker exec -it mongo-container mongosh "mongodb://user:password@host:port/?authSource=Database&readPreference=primary&appname=MongoDB%20Compass&ssl=false"
```
